# 💳 UPI Fraud Detection System

## 📖 Overview
A Machine Learning-based web application that predicts whether a UPI transaction is **Legitimate** or **Fraudulent** based on transaction details entered by the user.

## ✨ Features
- Detects fraudulent UPI transactions
- Instant prediction using Machine Learning
- Simple and user-friendly web interface
- Real-time prediction

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript

## ⚙️ How It Works
1. User enters transaction details.
2. Data is preprocessed.
3. The trained Machine Learning model analyzes the input.
4. The system predicts whether the transaction is **Fraudulent** or **Legitimate**.
5. The result is displayed on the webpage.
